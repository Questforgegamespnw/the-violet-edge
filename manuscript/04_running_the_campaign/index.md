# Running the Campaign

Bellweather works best when play moves between two scales:

> What does this person do?  
> What does the town do?

This section explains how to run that rhythm at the table.

The default version of the campaign uses Call of Cthulhu-style investigative horror with a light civic survival layer. Characters investigate, persuade, repair, flee, suffer stress, make mistakes, and bring clues home. Bellweather responds through councils, route offices, public notices, ration ledgers, retainers, records, and civic consequences.

The town is not a backdrop.

It is the campaign's second body.

This section covers:

1. **Call of Cthulhu Integration** — why CoC-style play fits Bellweather, how to use investigators, Luck, pushed rolls, injury, Sanity, Continuity Stress, and Signal Leaks.
2. **City Mode** — what town-scale play is, when to use it, how civic actions, authority, declarations, retainers, and public meaning shape the campaign.
3. **Expedition Mode** — how to run dangerous missions beyond stable Bellweather, including route declarations, return procedures, clues, drift, and recognition tests.
4. **Session Loop** — the repeatable table procedure: city briefing, council decision, Council Slots, civic rolls, resources, expedition play, upkeep, and return consequences.
5. **Alternate Game Systems** — how to adapt Bellweather to 5e/fantasy, Mothership/Violet Drift, and other engines without breaking the core premise.

## Reading Use

Read this section before the first session if you are the GM.

The most important pages are:

- **Call of Cthulhu Integration**, if you are using the default rules approach.
- **City Mode**, if you need to understand the town-scale layer.
- **Session Loop**, if you need the actual table procedure.

Use **Expedition Mode** once players begin leaving stable civic ground.

Use **Alternate Game Systems** only if you are converting the setting.

## Player-Facing Use

Some of this section can be summarized for players:

- Bellweather is fragile, and character choices affect the town.
- Civic decisions matter as much as investigation scenes.
- Public language can become dangerous.
- Expeditions need clear purpose, witnesses, routes, and return conditions.
- Retainers are people, not disposable spare lives.
- Failure should continue the story while changing cost, exposure, interpretation, or consequence.

The deeper Violet logic and GM procedures should remain GM-facing until learned through play.

## Running Principle

A strong session usually includes:

- one town-scale decision
- one focused mission or civic conflict
- one meaningful clue
- one human cost
- one Violet misinterpretation or response
- one consequence that changes the next session

Do not try to resolve everything.

Pick the pressure that matters now.

## Core Principle

> Start with Bellweather.  
> Choose what the town can bear.  
> Leave with purpose.  
> Return with consequence.
