How to Use This Book

This book is both a setting guide and a campaign engine.

The setting guide explains Bellweather, the Violet, the town’s institutions, public meaning, factions, locations, monsters, and major threats.

The campaign engine explains how to run the town under pressure: civic decisions, expeditions, public interpretation, trackers, trouble webs, and consequences.

The appendix provides compressed tools for table use after the GM has read the relevant sections.

Main Book vs. Appendix

The main book teaches judgment.

The appendix supports recall.

Read the main sections when learning how Bellweather works. Use the appendix during play when you need a tracker, procedure, handout, or one-page reference.

GM-Facing and Player-Facing Material

Some material is safe for players to read. Other material explains the Violet’s deeper logic, hidden trouble structures, faction secrets, and endgame doctrine.

Each wiki page uses player_safe metadata to help separate public-facing reference from GM-only truth.

How to Use Trouble Webs

A Trouble Web is not a quest. It is a civic-metaphysical problem with symptoms, hidden logic, factions, escalation, clues, and possible interventions.

Use a Trouble Web when you need a major pressure on Bellweather that can worsen, branch, and teach the Violet something.

How to Use Trackers

Trackers are not scoreboards. They are pressure instruments.

They show what the town is teaching the Violet, what resources are failing, what public fears are rising, and what kinds of consequences should become more likely.

The Violet Is Not a Gotcha Machine

The Violet should be used as a consequence engine, not a trap for casual table talk.

It responds most strongly to emotionally charged, repeated, public, ritualized, dream-borne, or reactive-zone meaning. Players should feel free to roleplay. The danger comes when meaning gathers weight.
